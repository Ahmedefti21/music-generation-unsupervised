# Music Generation Unsupervised

Unsupervised neural network project for music generation using the MAESTRO dataset.

## Mandatory Tasks
- Task 1: LSTM Autoencoder
- Task 2: Variational Autoencoder (VAE)
- Task 3: Transformer-based Music Generator

## Optional Task
- Task 4: Reinforcement Learning with Human Feedback

## Dataset
- MAESTRO MIDI dataset

## Project Structure
- `data/` raw and processed dataset files
- `notebooks/` notebooks for preprocessing and each task
- `src/` source code for preprocessing, models, training, and evaluation
- `outputs/` generated MIDI files, plots, and checkpoints
